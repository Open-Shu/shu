# Integrations Catalog (Active Document)
This document tracks current and potential integrations for Shu. Integrations are implemented as plugins; this list is intentionally incomplete and should be treated as a working catalog rather than a guarantee of support.



Currently supports
- Gmail (processing; domain-delegated)
- Google Calendar (processing; domain-delegated)
- Google Chat (processing; Admin SDK-backed participant resolution)
- Google Drive (processing)

Targets (selected)
- Communication/Calendars: Gmail, Outlook 365, Google Calendar, Exchange, Google Chat, Slack, Teams, Zoom, Meet
- Docs/Files/Knowledge: Google Drive/Docs, OneDrive/SharePoint, Dropbox, Box, Notion, Confluence, DocuSign
- Projects/Tasks: Jira, ClickUp, Asana, Linear, Monday.com, Trello, GitHub Issues
- CRM/RevOps: HubSpot, Salesforce, Pipedrive
- Eng/DevOps: GitHub/GitLab, PagerDuty, Sentry, Datadog
- HR/Identity: Admin SDK, Okta, Azure AD, BambooHR
- BI/Data: Sheets, Excel Online, Looker, Tableau, Snowflake, BigQuery
- Messaging/Notify: Slack/Teams cards, Twilio/SendGrid/MixMax
- Web/External: Web search (Google/Bing), RSS, custom HTTP sources
